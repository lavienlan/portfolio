Arrays.

What is an array?

An array is a collection of multiple values of the same data type that are stored in contiguous memory locations and can be accessed through a single variable. In Java, arrays can be of any type, including the primitive types (int, float, double), reference types (String) and object types.


Declaring an array

To declare an array in Java, there are 4 things you must specify: the data type, variable name, square brackets (indicates itâ€™s an array), and size. For example, to declare an integer array of size 5, you would use the following syntax:

int[] array = new int[5];

This creates an integer array with 5 elements, which are initialized to 0 by default.


Initializing an array

You can initialize an array in Java by specifying the values of its elements at the time of declaration. For example, to create an integer array with the values 1, 2, 3, 4, and 5, you would use the following syntax:

int[] myArray = {1, 2, 3, 4, 5};

Alternatively, you can initialize an array using a loop. For example, the following code initializes an array with the numbers 1 through 5: 

int[] myArray = new int[5]; 
for (int i = 1; i <= 5; i++) { 
     myArray[i] = i;
}

Accessing Array Elements


To access individual elements of an array in Java, you must specify the index of the element. Array indices start counting at 0, so the first element of an array is at index 0, the second element is at index 1, the fourth element is at index 3, and so on. Here is an example of what the syntax would look like if I wanted to access the fourth element of an array:

int[] myArray = {1, 2, 3, 4, 5};
int fourthElement = myArray[3];

This sets fourthElement to the value of the fourth element of myArray, which is 4.


Changing Array Elements

You can change the value of an array element in Java by assigning a new value to it. For example, to change the second element of an array to 10, we must use our previously learned knowledge about indices. Here we apply it in the following syntax:

int[] myArray = {1, 2, 3, 4, 5};
myArray[1] = 10;

This sets the second element of myArray to 10.


Array Length

You can determine the length of an array in Java using the length property (.length). For example, to get the length of an array, you would use the following syntax:

int[] myArray = {1, 2, 3, 4, 5};
int length = myArray.length;

This sets length to the length of myArray, which is 5.


Multidimensional Arrays

In Java, you can also create multidimensional arrays, or more commonly known as 2D arrays. These special arrays are essentially arrays of arrays.

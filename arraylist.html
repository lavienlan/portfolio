ArrayLists.

What is an ArrayList?

In Java, the ArrayList is a class used to implement the functionality of a dynamic/resizable array, which can grow or shrink automatically based on the number of elements present in it which provides a more flexible way of storing and manipulating data.


Creating an ArrayList

To create an ArrayList in Java, you need to import the java.util.ArrayList package and use the ArrayList class. Keep in mind that you can not use primitive data types for your ArrayLists and instead, must use the corresponding wrapper class (Integer, Double).

import java.util.ArrayList;
ArrayList animals = new ArrayList();

The code above creates an empty ArrayList of strings named 'animals'.
Now let’s take a look at some of the basic operations that can be done to an ArrayList.


Adding Elements to an ArrayList

To add elements to an ArrayList, you can use the add() method:

animals.add("Piglet");
animals.add("Rabbit");
animals.add("Mouse");

The above code adds three string elements to the ArrayList animals.


Accessing Elements of an ArrayList

You can access elements of an ArrayList using the get() method:

System.out.println(animals.get(0));
System.out.println(animals.get(1));
System.out.println(animals.get(2));
This code accesses the first, second, and third elements of the ArrayList animals using their respective indexes.


Removing Elements from an ArrayList

You can remove an element from an ArrayList using the remove() method:

animals.remove(1);

Removes the second element (Rabbit).


Modifying Elements of an ArrayList

You can modify an element of an ArrayList using the set() method:

animals.set(0, "Lion");

Changes the first element (Piglet) to Lion.


Size of an ArrayList

You can find the size of an ArrayList using the size() method:

System.out.println(animals.size());

Outputs the length: 3.


Iterating over an ArrayList

You can use a for loop to iterate over all the elements of an ArrayList: 

for (String x : animals) {
     System.out.println(x);
}
                    


                    This can be read as “for x in animals” and will iterate over all the elements 
                    of the Arraylist animals and print them to the console.
                

                

                
 Quick Conclusion 

                

                    In this tutorial, we learned about ArrayLists in Java. We saw how to create an 
                    ArrayList, add, access, remove, and modify elements of an ArrayList. We also 
                    learned how to find the size of an ArrayList and how to iterate over its elements. 
                    With its dynamic nature, ArrayList provides a more flexible approach of storing 
                    and manipulating data in Java and can hopefully help you in future code!
